# Hanuman Invitation Website

Official website for the Shree Abayanjaneya Swami Mandir Prana Pratishtha Mahotsavam.

## Features

* Hanuman-themed welcome page
* Beautiful flower animations
* Invitation card display
* Responsive design for mobile and desktop
* Event information for devotees and visitors

## Files

* `index.html` – Main website page
* `hanuman.jpg` – Hanuman image
* `invitation.jpg` – Invitation card image
